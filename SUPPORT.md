# Support

This file is the default support guide for repositories in the
`breezy-bays-labs` organization.

If a repository has its own `SUPPORT.md`, that repository-specific guide takes
precedence over this default.

## Where to Ask for Help

- Use GitHub Issues for reproducible bugs, documentation gaps, and concrete
  feature requests.
- Use GitHub Discussions, if enabled in the repository, for usage questions,
  design questions, and broader idea exploration.

## Security Issues

Do not report suspected vulnerabilities in public issues or discussions. Follow
the private reporting instructions in [SECURITY.md](./SECURITY.md).

## Conduct or Private Concerns

For Code of Conduct concerns or other private community issues, contact
`breezy-bays-labs@proton.me`.
