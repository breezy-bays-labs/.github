# Breezy Bays Labs

Tools and methodology for agentic software development — where AI agents are teammates, not assistants.

## How We Work

We adapt [Shape Up](https://basecamp.com/shapeup) for a team of **1 human + N concurrent AI agent sessions**. Structure IS memory — because agents start fresh every session, the methodology, pipeline, and knowledge systems provide the continuity that human teams take for granted.

**The human's three irreplaceable roles:**

| Role | When | What |
|------|------|------|
| **Bet** | Between cycles | Decide what to build next, based on appetite and strategic value |
| **Interview** | During shaping | Provide domain knowledge, validate requirements, make business decisions |
| **Smoke Test** | After build | Verify the built thing matches intent |

Everything between these touchpoints — research, shaping, breadboarding, planning, building, reviewing — is agent-executable.

## Principles

- **Schema-first** — Types are the source of truth, not afterthought documentation
- **Pipeline as methodology** — Every piece of work flows through defined stages; skip stages deliberately, never accidentally
- **Structure enables autonomy** — Clean taxonomy + structured templates + automated sync = agents that self-orient without hand-holding
- **Observation compounds** — Friction, patterns, and decisions captured during work become the knowledge that improves future work
- **GitHub as PM platform** — Issues, labels, milestones, and `gh` CLI are the coordination layer. No external sync tax.
